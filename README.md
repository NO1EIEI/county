# Zig-County 💖

เกมทายตัวเลขสุดคิวท์ 🚀 พัฒนาด้วยภาษา **Zig** และรันบน **Docker**

## ✨ คุณสมบัติ
- **3D Physics Engine:** ใช้ Three.js และ Cannon.js สำหรับเอฟเฟกต์การระเบิด
- **High Performance:** รันด้วย Zig Web Server แท้ๆ
- **Thai Localized:** ข้อความและคำใบ้น้อง OUMMY สุดน่ารัก

## 🚀 วิธีรันในเครื่อง (Local)
1. ติดตั้ง **Zig 0.15.2**
2. รันคำสั่ง `zig build run`
3. เปิดเว็บที่ `http://localhost:3000`

## ☁️ วิธี Deploy ฟรี
1. กดที่ปุ่ม **Deploy to Render** ด้านบน
2. เลือกแพ็กเกจ **Free** (ระบบจะเลือกให้อัตโนมัติจากไฟล์ `render.yaml`)
3. รอระบบคอมไพล์และรัน 2-3 นาที
